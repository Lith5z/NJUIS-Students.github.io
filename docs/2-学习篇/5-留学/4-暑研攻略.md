# 暑研攻略

暑研（暑期科研实习）一般安排在大三结束的暑假至大四上学期，前往海外教授的实验室进行较长期的科研实习。时长通常为 2–3 个月（短期）或 6 个月左右（长期）。

如果能联系到满意的学校和教授，**建议尽量争取 6 个月的长期暑研**，这样更有可能拿到强推荐信或 return offer。暑研本质上更像是自费去给老师打工，所以难度不一定特别高，很多招人不多的教授可能是先到先得，或者哪天刚好想起来看邮箱，就招一两个学生过去。有时候看运气。

但也要注意，招人很多的老师不一定更好。你很可能只是进了一个大池塘，和教授见面、说话的机会都比较少，更不用说 return offer 了。大概率要卷到所有实习生里的头部，才有可能拿到真正有分量的推荐信或后续机会。所以这些最好提前打听清楚。

本文大致分为五部分：

- 第一部分讲为什么要做暑研；
- 第二部分介绍南大本校合作项目；
- 第三部分讲自主套磁的完整流程；
- 第四部分讲如何判断一个暑研机会值不值得去；
- 第五部分补充一些最后建议。

---

## 一、为什么要做暑研

1. **助力留学申请：** 对于想申请美国直博或 CS 强校热门硕士项目的同学来说，暑研是获取教授推荐信或 return offer 的核心途径。尤其对于申美的同学，美国学术界和高校普遍不太认可国内教授的推荐信，一封来自美国教授的强推往往具有决定性作用，connection is all you need！
2. **提前适应海外环境：** 深入了解和适应美国的文化、生活和科研氛围。这些方面与国内存在不小的差异，提前体验有助于未来留学时更快融入。
3. **拓展研究视野：** 接触更前沿、更广阔的研究方向，也有机会产出论文成果。如果后续能继续合作，暑研就不只是暑假体验，而是申请中非常硬的一段经历。

## 二、本校合作项目

南大提供美国暑研奖学金项目，同学们可以关注。以下是大致的时间线：

| 时间 | 事项 |
|------|------|
| 大三上 9–10 月初 | 开放申请 |
| 10 月末 | 资料提交 + 面试 |
| 11 月 | 教授录取反馈 |
| 12 月 – 次年 2/3 月 | 与教授自行联系，走对方学院流程，获取 DS-2019（相当于官方录取函） |
| 3–6 月 | 办理美签（J-1 签证），越早预约和面签越好，存在被拒或被 check 的可能。签证拿下后即可开始租房、办信用卡、订机票等 |
| 7 月 | 一般 7 月左右前往线下暑研 |

**一些 Tips：**

- 南大合作项目的教授大多为本校校友（在美做教职），申请难度相比自己套磁会小不少。也有部分清华本科背景的老师，要求会更高一些。
- 有的老师会额外安排面试或考核，有的则直接根据材料和自我介绍做出选择。
- 选择教授时务必仔细浏览其主页和近期论文。同时关注组内是否有本科生或实习生。一方面，有些教授不习惯指导本科生，缺乏相关经验；另一方面，如果组内实习生较多，竞争强推或 return offer 的压力也会更大。

## 三、自主套磁怎么做？

如果在南大提供的暑期研究项目中没有找到与研究方向匹配的教授，可以选择**自主套磁**。所谓套磁，是指主动向海外大学的教授发送邮件，自我介绍并询问是否可以前往其课题组进行科研实习（Research Intern / RA）。

实习形式通常为线下，周期一般为 2–6 个月，也有少量远程形式。整体流程与通过学校项目申请暑研基本一致，不同之处在于需要**自行寻找导师并主动联系**，而不是从学校提供的导师列表中选择。

### 1. 如何寻找导师

**社交平台信息**

在小红书、知乎等平台上，有时会有教授或实验室成员发布招收 Research Intern / RA 的信息。可以根据帖子中的联系方式进行申请。

**学术排名网站**

可以在 [CSRankings](https://csrankings.org) 上筛选美国高校，并按照研究方向查找相关教授。进入教授个人主页后，查看是否有以下信息：

- 是否招收 Research Intern / RA；
- 是否提供申请表（例如 Google Form）；
- 是否需要额外发送邮件说明申请。

如果主页提供申请表，一般需要先填写表格，再发送邮件说明申请情况。

**阅读论文寻找导师**

通过最近的顶会论文（如 CVPR、ICCV、NeurIPS、ICML 等）找到感兴趣的研究方向，再查看论文作者所在实验室主页，确认是否接受实习生。

**connection 推荐**

大多数国内老师其实都会认识一些海外导师。如果你很早就在国内组里开始做科研，并且做得整体不错，可以问问老师有没有 connection，能不能直接帮忙内推。这往往会比自己海套舒服很多，也不用担心太多信息差问题。

### 2. 套磁邮件写什么

套磁邮件通常比较短，核心是让教授快速判断：你是谁、你想做什么、你和他的方向是否匹配、你有没有值得进一步看的经历。

邮件一般包含以下信息：

- **基本信息：** 来自哪所学校、什么专业、当前年级；
- **申请意图：** 希望在某个时间段前往课题组进行 Research Intern / RA；
- **研究经历：** 参与过哪些科研项目，研究内容是什么，是否有论文或技术成果；
- **研究方向匹配：** 说明自己的研究方向与教授的某些研究工作或论文方向相匹配；
- **附件材料：** 在邮件中附上个人简历（CV），必要时附上论文、项目或 GitHub 链接。

邮件本质上是对简历内容的简要概括。不要在邮件里写小作文，教授如果感兴趣，会通过简历获取更完整的信息。

### 3. 套磁数量和节奏

由于回复率通常较低，建议同时联系 **20–30 位教授**。说白了就是海套，不要太优雅，先把量打够。

在选择导师时，不必只局限于华人教授，可以同时联系不同国家背景的导师，以提高成功率。热门方向回复率会更低，冷门交叉方向反而可能更容易拿到机会。

时间上，多数实验室在 **1–3 月** 就会基本确定暑期实习名额。如果你等到四五月才开始套，当然也不是完全没机会，但会被动很多。

### 4. follow-up 怎么发

若 7–10 天没有回复，可以发送一次简短 follow-up 邮件。这个很重要，本人就是 follow-up 之后才收到面试通知，最后拿到暑研机会的。

follow-up 不需要写很长，大概表达三件事就够了：

- 简单提醒自己之前发过申请；
- 再次表达对该方向 / 课题组的兴趣；
- 礼貌询问是否有暑期 Research Intern / RA 的机会。

不要一天一催，也不要隔太久才想起来。一般一封原始邮件 + 一封 follow-up 就够了，如果还没回复，就可以把精力放到其他老师身上。

### 5. 邮箱和材料准备

建议使用 **Gmail 或 Outlook** 发送套磁邮件。

部分学校邮箱（例如 `@nju.edu.cn`）在向国外高校邮箱发送邮件时，可能会因为邮件服务器信誉或过滤策略，被对方邮箱系统自动识别为垃圾邮件，从而进入 **Spam / Junk** 文件夹，导致教授无法及时看到邮件。

使用 Gmail 或 Outlook 发送邮件通常具有更高的送达率。

提前准备好这些材料：

- CV（科研经历为主）；
- 成绩单；
- GitHub / Google Scholar / 个人主页；
- 已发表或在投论文；
- 一个简短清晰的研究兴趣总结。

## 四、如何判断一个暑研机会值不值得去？

拿到暑研机会之后，不要只看学校名气。暑研真正有用的地方，往往取决于导师是否认真带、项目是否匹配，以及后续能不能产生推荐信、合作或 return offer。

可以重点看下面几件事：

1. **导师是否真的带本科生或者是否会安排博士带。** 有些教授很强，但不一定习惯带短期本科实习生。最好看看组内过去有没有 visiting student、本科生或暑研学生。
2. **组内实习生数量有多少。** 如果一个暑假招很多人，你就要小心自己是不是进了池塘。人多不一定不好，但教授平均分给每个人的时间会明显变少。
3. **项目是否足够具体。** 如果老师能给出比较明确的方向、问题和预期产出，通常比“来了再说”更靠谱。
4. **是否有继续合作的可能。** 暑研结束后能不能远程继续做？有没有机会投论文？有没有机会拿强推荐信？这些都比项目 title 本身更重要。
5. **线下 / 远程、时长和成本。** 线下体验更完整，也更容易和老师、组内同学建立关系；远程成本低，但存在感和推进效率往往差一些。时长上，6 个月通常比 2–3 个月更有机会做出实质成果。
6. **导师风格和组内氛围。** 能提前问到组内学生最好。导师 push 不 push、会不会定期 meeting、组内有没有合作氛围，之前来这边做暑研的同学最后申请到了哪里？

---

## 五、最后建议

通过自主套磁获得暑研机会通常需要一定数量的尝试和持续沟通。如果没收到面试通知，那大概率就是套得还不够多。
